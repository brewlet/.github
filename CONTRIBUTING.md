# Contributing to Brewlet

Thanks for helping improve Brewlet.

Before starting substantial work, open an issue in the repository that owns the
change so the approach and scope can be discussed. Keep pull requests focused,
describe the motivation and behavior change, and include relevant tests and
documentation.

For the core runtime, operator, CLI, and general project documentation, start in
[`brewlet/brewlet`](https://github.com/brewlet/brewlet).
